# Zmaga Cigli

Official website for **Zmaga Cigli** — a company specializing in decorative 
gypsum bricks for interior use, offering a premium collection of wall solutions 
for modern and refined spaces.

🌐 [zmagacigli.com](https://zmagacigli.com)

## About

This is the official presentation website for Zmaga Cigli, showcasing their 
product catalog, collections, and contact information. The site is designed 
to provide a seamless and elegant browsing experience.

## Features

- 🧱 Product catalog with detailed specifications
- 📐 Collection showcase with high-quality imagery
- 📱 Fully responsive design for all devices
- ⚡ Fast performance with smooth scrolling
- 📩 Contact section for inquiries

## Tech Stack

- [Next.js](https://nextjs.org) — React framework
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [Vercel](https://vercel.com) — Hosting & deployment
